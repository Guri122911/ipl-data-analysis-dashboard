IPL Data Analysis and Dashboard (2008–2018)
Project Overview

This project analyzes Indian Premier League (IPL) match data from 2008 to 2018. The dataset contains 697 matches and 15 columns, including match details such as teams, toss decisions, venues, match winners, and player performance. The goal of this project is to analyze IPL data to identify patterns and insights such as the impact of toss decisions, batting vs fielding first, team performance, venue analysis, and player performance.
